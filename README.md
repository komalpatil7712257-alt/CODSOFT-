# CodSoft_TIC_TAC_TOE--Python
This is a simple Tic Tac Toe game made using Python.
It is a two player game where players take turns to enter their moves on a 3x3 grid.
# Features
- Two player game
- Shows winner or draw
- Easy to play
# How to run
Run the python file and follow the instructions.
# Language Used
Python
